移动端的chrome浏览器效果更佳 : )

打开方式：

1 将项目 clone 下来 
```
$ https://github.com/Siwen19/Geek.git
$ cd geektime-frontend
$ npm install
```
2 运行 

打开后端提供的数据
```
$ cd geektime-backend 
$ node index.js 
```
打开前端
```
$ cd geektime-frontend 
$ npm start
```
现在就在本地的3000端口访问了,其中后端提供的数据可在8080端口
